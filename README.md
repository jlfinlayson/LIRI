# LIRI

LIRI is a command line application that allows you to for information on your favorite movies, songs and bands. There is even a random feature if you are unable to decide on what to search for. <br>

 
 ### Technologies Used

* Node.js
* Javascript
* Axios
* NPM
* Spotify API
* Moment.js
* OMDB API
* Bands In Town API
* dotenv
